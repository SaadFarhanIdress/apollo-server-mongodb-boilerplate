# Config file ðŸ’»
Go the config file src/config_prod.js. You will see:
```javascript
module.exports = {
    DATABASE_URL: "YOUR MONGODB URL"
}
```
Replace "YOUR MONGODB URL" with your MongoDB URI.
Rename it to config.js.
Install dependencies.
```bash
npm i 
OR
yarn
```
Start the server.
```bash
npm start
```
